# for 文

基礎的な for 構文を書いてください。

##

```js
function setup() {
    // ここに処理を記述
}
```

# for 文による 1~10 までの数字の出力

1 から 10 までの数字を出力する for 文を書いてください。

##

```js
function setup() {
    for () {
        print();
    }
}
```

# for 文による 偶数の出力

1 から 10 までの数字のうち、偶数の数字のみを出力する for 文を書いてください。

##

```js
function setup() {
    for () {
        if () {
            print();
        }
    }
}
```

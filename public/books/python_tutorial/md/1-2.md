# Jupyter Notebookとは？

Jupyter Notebook は、Python コードを対話的に実行できる環境です。

## VSCode で Jupyter Notebook をはじめる

### ノートブックの作成

1. VSCode を起動し、プロジェクトの `python` フォルダを開く。
   ![](/books/python_tutorial/img/1-3/0.png)
2. 左サイドバーの**New Fileアイコン** をクリックする。
   ![](/books/python_tutorial/img/1-3/1.png)
3. ファイル名を `01_test.ipynb` として保存。
   ![](/books/python_tutorial/img/1-3/2.png)

4. 作成したファイルをクリックして、ノートブックを開きます。その後「a」キーを押すと、ノートブックの下に新しいセルが追加されます。
   ![](/books/python_tutorial/img/1-3/3.png)

5. セルに`print("こんにちは！")`と入力し、`Shift + Enter`で実行します。
   ![](/books/python_tutorial/img/1-3/4.png)
6. まだPyhonの実行カーネルを選択していない場合、画像のようなダイアログが表示されます。「???」と言うPythonの実行カーネルを選択してください。
7. 再度`Shift + Enter`を押すと、セルが実行され、出力結果がセルの下に表示されます。

おめでとうございます🎉🎉🎉これで初めてのコードを実行できました。

### セルの使い方

ノートブックは、セルと呼ばれる小さなブロックで構成されています。各セルにはコードを入力できます。

| ショートカット  | 説明                           |
| --------------- | ------------------------------ |
| `a`             | 上に新しいセルを追加           |
| `b`             | 下に新しいセルを追加           |
| `x`             | セルを削除                     |
| `Shift + Enter` | セルを実行し、次のセルに移動   |
| `Ctrl + Enter`  | セルを実行し、同じセルに留まる |
